# 프로그램 목록

## 1. 문서 목적

- 현재 코드 기준의 실제 프로그램/모듈 경로를 정리한다.
- 기능별 책임과 진입점을 빠르게 찾을 수 있게 한다.
- 예전 `routers/services/repositories` 기준이 아니라 현재 `domains/*` 구조를 기준으로 작성한다.

## 6. 참고

- 현재 구조의 중심은 `backend/domains/*`와 `frontend/src/features/*`입니다.
- 예전 `backend/routers/*`, `backend/services/*`, `backend/repositories/*` 분류 기준은 더 이상 기준 문서로 사용하지 않습니다.
