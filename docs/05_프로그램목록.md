# 프로그램 목록

## 1. 문서 개요

- 시스템명: RAG기반 법률 상담 및 문서 요약 시스템
- 문서 목적:
  - 시스템을 구성하는 주요 프로그램, 모듈, 화면, API를 정리한다.
  - 개발 범위와 역할 분리를 명확히 한다.
  - 유지보수 시 참조 가능한 프로그램 인벤토리를 제공한다.

## 8. 비고

- 본 문서는 최종 목표 기능 기준으로 작성된 프로그램 목록 초안이다.
