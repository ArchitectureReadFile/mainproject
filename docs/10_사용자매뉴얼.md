# 사용자 매뉴얼

## 1. 문서 개요

- 시스템명: RAG기반 법률 상담 및 문서 요약 시스템
- 문서 목적:
  - 최종 사용자가 시스템의 주요 기능을 스스로 사용할 수 있도록 안내한다.
  - 회원가입, 로그인, 법률 상담, 문서 업로드, 워크스페이스 협업 절차를 설명한다.
- 대상 사용자:
  - 비로그인 사용자
  - 일반 사용자
  - 프리미엄 사용자
  - 워크스페이스 멤버

## 2. 서비스 소개

본 시스템은 법률 상담과 문서 요약을 지원하는 웹 기반 서비스이다.

사용자는 질문을 입력하여 관련 법률 및 판례 정보를 기반으로 한 답변을 받을 수 있으며, 워크스페이스를 통해 PDF 문서를 업로드하고 AI 요약 결과를 확인할 수 있다. 또한 멤버 초대, 문서 승인, 댓글 협업 기능을 통해 팀 단위 문서 검토가 가능하다.

## 3. 접속 방법

### 3.1 서비스 접속

1. 웹 브라우저를 실행한다.
2. 서비스 URL에 접속한다.
3. 메인 화면에서 챗봇 또는 로그인 기능을 사용할 수 있다.

### 3.2 권장 환경

- Chrome 최신 버전
- Edge 최신 버전
- Safari 최신 버전

## 4. 회원가입 및 로그인

### 4.1 회원가입

1. 메인 화면 우측 상단의 로그인 또는 회원가입 버튼을 클릭한다.
2. 회원가입 탭으로 이동한다.
3. 이메일 주소를 입력하고 인증번호 발송 버튼을 클릭한다.
4. 이메일로 수신한 인증번호를 입력하고 인증을 완료한다.
5. 이름과 비밀번호를 입력한다.
6. 회원가입 버튼을 클릭한다.

주의사항:

- 이메일 인증이 완료되어야 회원가입이 가능하다.
- 비밀번호는 보안 규칙에 맞게 입력해야 한다.

### 4.2 로그인

1. 로그인 모달을 연다.
2. 이메일과 비밀번호를 입력한다.
3. 로그인 버튼을 클릭한다.
4. 로그인 성공 시 메인 화면 또는 이전 화면으로 이동한다.

### 4.3 로그아웃

1. 헤더 또는 마이페이지에서 로그아웃 버튼을 클릭한다.
2. 업로드 중인 작업이 있다면 확인 메시지를 확인한다.
3. 로그아웃이 완료되면 메인 화면으로 이동한다.

## 5. 법률 상담 챗봇 사용 방법

### 5.1 기본 질문하기

1. 메인 화면의 채팅 입력창에 질문을 입력한다.
2. 전송 버튼을 클릭한다.
3. 시스템이 관련 법률 및 판례 정보를 검색한 뒤 답변을 생성한다.
4. 답변 영역에서 참고 근거와 출처를 확인한다.

예시:

- 계약 해지 시 손해배상 책임은 어떻게 판단되나요?
- 회사 내부 규정과 근로계약 충돌 시 어떤 기준이 우선하나요?

### 5.2 로그인 상태에서 채팅 사용

로그인 사용자는 다음 기능을 추가로 사용할 수 있다.

- 채팅 세션 저장
- 이전 대화 목록 조회
- 세션 제목 수정
- 세션 삭제

### 5.3 참고 문서와 함께 질문하기

1. 채팅 세션에서 문서 첨부 기능을 선택한다.
2. PDF 문서를 첨부하거나 워크스페이스 문서를 선택한다.
3. 질문을 입력한다.
4. 시스템은 첨부 문서를 참고하여 더 정확한 답변을 생성한다.

## 6. 워크스페이스 사용 방법

### 6.1 워크스페이스 목록 확인

1. 로그인 후 워크스페이스 메뉴로 이동한다.
2. 내가 속한 워크스페이스 목록을 확인한다.
3. 원하는 워크스페이스 카드를 클릭하여 상세 화면으로 이동한다.

### 6.2 워크스페이스 생성

1. 워크스페이스 목록 화면에서 생성 버튼을 클릭한다.
2. 워크스페이스 이름과 설명을 입력한다.
3. 생성 버튼을 눌러 워크스페이스를 만든다.

### 6.3 멤버 초대

1. 워크스페이스 상세 화면으로 이동한다.
2. 멤버 관리 영역에서 초대 기능을 선택한다.
3. 초대할 사용자 정보와 권한을 입력한다.
4. 초대 요청을 전송한다.

권한 구분:

- OWNER: 워크스페이스 전체 관리
- ADMIN: 문서 및 멤버 관리
- EDITOR: 문서 업로드 및 작업 참여
- VIEWER: 조회 중심 권한

## 7. 문서 업로드 및 요약 확인

### 7.1 문서 업로드

1. 워크스페이스 상세 화면에서 업로드 화면으로 이동한다.
2. PDF 파일을 드래그 앤 드롭하거나 파일 선택 버튼을 클릭한다.
3. 필요 시 승인 담당자를 선택한다.
4. 업로드 및 요약 생성 버튼을 클릭한다.

### 7.2 업로드 상태 확인

업로드 후 문서 상태는 다음과 같이 표시된다.

- PENDING: 처리 대기
- PROCESSING: 요약 처리 중
- DONE: 요약 완료
- FAILED: 처리 실패

### 7.3 요약 결과 확인

1. 문서 목록에서 원하는 문서를 선택한다.
2. 문서 상세 화면에서 요약 결과를 확인한다.
3. 필요 시 요약 PDF 다운로드 버튼을 클릭한다.

## 8. 문서 협업 기능 사용 방법

### 8.1 댓글 작성

1. 문서 상세 화면을 연다.
2. 댓글 패널을 연다.
3. 일반 댓글 또는 검토 댓글을 선택한다.
4. 댓글을 입력한 뒤 등록한다.

### 8.2 멘션 사용

1. 댓글 입력창에서 `@`를 입력한다.
2. 멘션할 사용자를 선택한다.
3. 댓글 등록 시 해당 사용자는 멘션 대상으로 기록된다.

### 8.3 승인 및 반려

승인 권한이 있는 사용자는 다음을 수행할 수 있다.

- 문서 승인
- 문서 반려
- 반려 사유 입력

## 9. 마이페이지 사용 방법

### 9.1 내 정보 확인

마이페이지에서는 다음 정보를 확인할 수 있다.

- 닉네임
- 이메일
- 구독 상태
- 보안 설정
- 알림 설정

### 9.2 계정 정보 변경

사용자는 마이페이지에서 다음 정보를 변경할 수 있다.

- 닉네임
- 이메일
- 비밀번호

### 9.3 구독 관리

프리미엄 구독이 필요한 기능은 구독 상태에 따라 접근이 제한될 수 있다. 마이페이지에서 구독 시작 또는 해지 상태를 확인할 수 있다.

## 10. 알림 사용 방법

### 10.1 알림 확인

알림 아이콘을 클릭하면 다음 항목을 확인할 수 있다.

- 워크스페이스 초대
- 문서 승인 요청
- 답변 완료
- 삭제 예정 안내

### 10.2 알림 처리

사용자는 다음 기능을 사용할 수 있다.

- 개별 알림 읽음 처리
- 전체 알림 읽음 처리
- 알림 삭제
- 알림 설정 변경

## 11. 자주 발생하는 문제와 해결 방법

### 11.1 로그인 실패

- 이메일과 비밀번호를 다시 확인한다.
- 인증되지 않은 계정인지 확인한다.
- 여러 번 실패한 경우 잠시 후 다시 시도한다.

### 11.2 문서 업로드 실패

- PDF 파일인지 확인한다.
- 파일 용량 제한을 초과하지 않았는지 확인한다.
- 네트워크 상태를 확인한 뒤 다시 시도한다.

### 11.3 요약 결과가 보이지 않음

- 문서 상태가 `PROCESSING`인지 확인한다.
- 일정 시간이 지난 뒤 다시 조회한다.
- `FAILED` 상태인 경우 다시 업로드하거나 관리자에게 문의한다.

### 11.4 챗봇 답변이 기대와 다름

- 질문을 더 구체적으로 입력한다.
- 참고 문서를 첨부한 뒤 다시 질문한다.
- 워크스페이스 문서 범위를 좁혀서 재질문한다.

## 12. 유의사항

- 본 서비스의 답변은 참고용 정보이며, 최종 법률 판단을 대체하지 않는다.
- AI 응답은 모델과 검색 결과에 따라 달라질 수 있다.
- 중요 문서는 업로드 전 내부 보안 정책을 반드시 확인해야 한다.
