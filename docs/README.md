## 2. 정리 원칙

- `docs/` 루트에는 현재 기준 문서만 둔다.
- 초안, 구버전, 비교용 문서는 별도 보관하지 않고 정리 대상이면 제거한다.
- 같은 목적의 문서가 여러 개 생기지 않도록 문서 번호와 역할을 고정한다.
- 테스트 문서는 현재 자리만 맞춰두었으며, 실제 테스트 기준으로 계속 보완한다.
